This repository contains a collection of Python projects focused on financial modelling and basic data analysis. The goal of these projects is to practice Python fundamentals while applying them to real-world financial and business scenarios.

## Projects

### Loan Calculator
Calculates monthly loan repayments using the loan amortization formula.

Features:
- Monthly payment calculation
- Total repayment calculation
- Total interest paid

Concepts used:
- User input
- Mathematical formulas
- Conditional logic
- Formatted output

---

### ROI Calculator
Calculates Return on Investment (ROI) and determines whether the investment resulted in a profit, loss, or break-even.

Concepts used:
- Arithmetic calculations
- Percentage calculations
- Conditional logic

---

### Cash Flow Projection
Projects the future balance of a business over multiple months by calculating income minus expenses.

Concepts used:
- Loops
- Financial forecasting
- User input
- Basic modelling

---

### Sales Data Analysis
Analyzes a dataset of sales values and calculates key descriptive statistics.

Statistics calculated:
- Total sales
- Mean
- Median
- Minimum value
- Maximum value
- Standard deviation

Concepts used:
- Lists
- Python statistics module
- Basic data analysis

---

### CSV Sales Data Analysis
Loads sales data from a CSV file and performs statistical analysis.

Concepts used:
- File handling
- Reading CSV data
- Data cleaning
- Statistical calculations

---

### Sales Trend Analysis
Analyzes sales data over time and identifies the overall trend.

Features:
- Calculates total change
- Calculates average change per period
- Detects upward, downward, or stable trends
- Displays period-by-period changes

Concepts used:
- Loops
- List indexing
- Conditional logic
- Time-series style analysis

---

### Growth Rate Calculator
Calculates the percentage growth between two values.

Formula used:

Growth Rate = (Final Value − Initial Value) / Initial Value × 100

Features:
- User input
- Percentage growth calculation
- Interpretation of growth results
