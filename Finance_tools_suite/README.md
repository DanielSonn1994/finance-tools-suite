# Finance Tools Suite

This repository contains a collection of Python financial and data analysis tools.

## Tools Included

**Loan Calculator**

* Calculates monthly loan payments
* Shows total repayment and interest

**ROI Calculator**

* Calculates return on investment
* Identifies profit, loss, or break-even

**Cash Flow Projection**

* Forecasts monthly balance over time

**Sales Analysis**

* Calculates statistics from a sales dataset
* Mean, median, standard deviation, min and max

## Dataset

`sales_data.csv` contains example daily sales data used for analysis.

## Concepts Used

* Python variables
* User input
* Loops
* Conditional logic
* File handling
* CSV data processing
* Basic statistics
